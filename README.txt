USAGE:
1. Change directory to SeattleTrails folder.
2. Run server: py server.py
3. Server can be quit by signaling CTRL+C


OVERVIEW:
This project implements the python Flask framework. 
It contains basic functionality for user registration and authentication.
Additionally, it supports Create, Read, Update, Delete functionality. The project
is currently in its preliminary stages of development.
